The heart of the forge

One Codex. Every tool.

The Codex is your single source of truth — every name table, spell, magic item, class, and stat block lives there, and every tool reads from it live: the generators, the character builder, the sheets, the tracker.

Add your homebrew once and it appears everywhere — in loot, in shops, in the builder's class list. Press Ctrl K anywhere to search all of it in an instant.

Export your whole library to one file to back it up or hand to another GM — or publish it to your campaign, and your players' tools pull it automatically.

Yours, everywhere

Install It Like an App

ArcanaForge is a Progressive Web App — open it in Chrome or Edge and choose Install from the address bar. It runs on desktop, tablet, and phone.

⚒

Works Offline

Once installed, the whole toolkit runs without a connection — basement game rooms and dead-zone cabins included.

🛡

Your Data Stays Yours

Solo by default — everything lives on your own device, no account, nothing uploaded. Link a campaign and only the shared tools sync, through a secure backend, so the table can play together.
